# 灾害及时推送与天气卡片 - AstrBot 插件

基于官方数据，主动推送国内地震、台风、海啸/海洋、极危红色天气；并支持 `/天气 北京` 按需查询天气卡片。

## 功能

### 主动及时推送
- 地震（中国地震台网 CEIC，仅国内）
- 台风（中央气象台台风网，仅国内相关；近岸 + 风力阈值可配，默认 8 级）
- 海啸/海洋（中央气象台相关预警）
- 极危红色天气（暴雨/台风/山洪等红色高危；同省同类型地区扩大时用「新增地区」文案）
- 四类推送可在 WebUI **分别开关**
- 可选夜间静默：静默时段仅放行地震
- 推送文案附带安全忠告，不输出来源链接/数据源名

### 不主动推送
- 普通橙/黄/蓝极端天气

### 天气查询
- 命令：`/天气 北京`、`/天气 成都`
- 支持市、以及中央气象台有站点的区/县
- 成功时直接返回卡片图，不发“正在查询”和文字简讯

## 安装

1. 将本目录放到 AstrBot：`data/plugins/astrbot_plugin_disaster_alert`
2. WebUI 启用插件
3. 配置目标群 `target_groups`
4. 如缺 Pillow：在 AstrBot 使用的 Python 环境执行  
   `pip install -r requirements.txt`

## 配置建议

| 项 | 建议 |
|---|---|
| 地震推送 | 开，最小震级 4.5 或 5.0 |
| 台风推送 | 开，最低风力 8 级 |
| 海啸/海洋推送 | 开 |
| 极危红色天气 | 开（默认发图片，失败降级文字） |
| 夜间静默 | 可选，如 23:00-07:00 仅放行地震 |
| 轮询间隔 | 120 秒 |
| 启动跳过历史 | 开 |
| 每轮最多推送 | 5~8 |

目标群示例：
- `123456789`
- `aiocqhttp:GroupMessage:123456789`

## 命令

| 命令 | 说明 |
|---|---|
| `/天气 北京` | 查询天气卡片 |
| `灾害状态` | 查看开关、静默、数据源健康 |
| `灾害检测` | 立即拉取摘要（不推送） |
| `灾害推送` | 立即检测并推送到配置群 |
| `最近预警` / `灾害记录` | 查看近期已成功推送记录 |
| `灾害帮助` | 帮助 |

## 推送形态

- 地震 / 台风 / 海啸海洋：纯文字
- 极危红色天气：默认图片，失败才文字
- 台风距陆距离文案带「约 / 估算」

### 台风示例

```text
【台风动态】台风 巴威 最新动向
时间：2026年07月14日10时00分
等级：热带风暴（近中心风力8级，18 m/s）
实况：中心气压 992 hPa；移向移速 NE 34 km/h
中心位置：北纬34.7°，东经118.7°（距大陆最近约 40 公里，估算）
影响范围：山东省·烟台市、山东省·威海市、山东省·青岛市

【安全忠告】...
```

## 验证

1. 配置目标群并启用插件
2. 群内发送：`灾害状态`
3. 群内发送：`/天气 北京`，应直接收到卡片
4. 发送：`灾害检测`，应只看到及时通道事件
5. 发送：`最近预警`，查看已推送历史
